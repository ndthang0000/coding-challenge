## Description

This project demonstrates how to manage sessions and tasks using BullMQ and MongoDB. The following features are included:

1. Create a session via BullMQ and store it in MongoDB. The ```status``` of the session will be change realtime based on list task status
2. Retrieve a list of sessions.
3. Get detailed information about a session, including its associated tasks.
4. Create a task within a session that automatically completes after a specified time (`ms`) based on `timeInspect`.

## Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
- Install [Git](https://git-scm.com/) for version control.
- Ensure you installed Redis on your machine
